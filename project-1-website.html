Welcome to My First Project

This is my very first project, you are looking at it right now! Its is a simple portfolio website that I created at the start of Year 9 to learn web development!

What Does It Include?

My first project was designed to show off my skills in web development. It includes:

  • A simple header and footer consistent across all pages
  • Written in HTML and CSS
  • Cards for each project with a title and description
  • Hover elements such as box shadow and transform
  • Dropdown menu for the project links in the header
  • Responsive design that works on different screen sizes

My Reflection

I am very happy with my first web development project turned out, it has allowed me to apply my skills practically.

This project was a great introduction to web development and helped me understand the basics of HTML and CSS. I had to apply my knowledge from learning in class to create a responsive and appealing website.

Visual Code Studio was very helpful, mainly in debugging the code as it provides error detection as you are writing code. Git was also useful because If i made a mistake it could easily be reverted to a previous version. GitHub was necessary to share my code so it could be accessed by others.

When designing my portfolio, I changed many of my initial ideas as I thought of new and better ones. The website allowed me to develop my skills in CSS and because of this I learned new CSS properties and customised my website in ways that I didn't know before. This includes things such as box shadow and transform to bring HTML elements to life when hovered over for example. Another example is background attachment to create a cool parallax design when scrolling down my website.

Some challenges I faced were designing and implementing buttons, this is because I didn't know what they were at all and how to style them with CSS and they are quite different from the default text or links in HTML. However I quickly learned that it was quite easy because you can just add a class and change the dimensions of these buttons to fit certain needs.

In the future, I would like to add more advanced features and improve on existing features. I would also like to add JavaScript when I learn about it in the future. I would also like to experiment with different styles such as the colour palette and typography.

This project has taught me quite a lot about HTML and CSS and especially reinforced the fact that websites are made up of boxes, made with HTML elements such as divs [which I also learned lots about and practiced using them in many different scenarios]. It also taught me more about CSS and how it was connected to HTML with things like class selectors.
